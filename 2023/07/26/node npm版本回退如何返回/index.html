node npm版本回退 如何恢复npm最高版本

发表于2023-07-26更新于2024-06-30阅读次数

网上的方法大多是两种

笔者使用过后都没有达到预期效果（估计是回退版本太多了）

笔者的方法：

1.重新安装nodejs

2.使用命令‘where node’ ‘where npm’找到两者位置

3.根据返回的位置，删除掉之前安装的nodejs和npm

然后重新安装就大功告成啦
